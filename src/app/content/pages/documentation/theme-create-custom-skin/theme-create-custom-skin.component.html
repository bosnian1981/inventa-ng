Create a Custom Skin?

The prosess of creating a Custom Skin is very simple, because _custom.scss file has been already created and all you need to do is to follow these steps:

  • 1. Change the line: skin: 'light-blue' to skin: 'custom' in settins.service.ts file located at app/content/layout/_settings/
  • 2. Change styles in _custom.scss file located at app/styles/skins/
